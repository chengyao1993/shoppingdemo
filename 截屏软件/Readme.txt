========================================================
    Setuna Easy Scrapping   ver 1.5.5
========================================================

  [Setuna Easy Scrappingとは？]
    ボタン一つで画面の一部を切り抜き、最前面に表示させておくソフトです。
    作業中に他のアプリケーションの一部を参照したい場合等に使用できます。
    
    
  [確認済み動作環境]
    Windows XP
    Windows 2000
    
    
  [インストール]
    インストーラーは使用していませんので、Setuna.exeを実行するだけです。
    
    
  [アンインストール]
    そのまま削除してください。
    
    
  [使い方]
    1.Setuna起動後、Ctrl+1で画面のスクリーンショットを表示します。
      （スクリーンショット表示中はマウスカーソルが変更されます。）

    2.マウスをドラッグし、切り抜きたい範囲を選択します。

    3.スクラップが作成されます。

    作成されたスクラップは常に最前面に表示されます。マウスでスクラップの
    移動も可能です。
    
     * ショートカットキーのデフォルト設定は Ctrl+1です。
       他のソフトで予約されている場合は使用できません。
       その場合はオプションからショートカットキーを設定しなおす必要があります。
    
  [機能]
    プリセットスクラップ
       ◆スクラップ登録
         Ctrl+1でスクリーンショットを表示し、Shift+1〜4を押すと番号に対応した箇所に予約を入れます。
         通常と同じように範囲を選択すると、確定となります。
       
       ◆スクラップ呼び出し
         Ctrl+1でスクリーンショットを表示し、1〜4を押すとあらかじめ登録したスクラップの範囲を切り抜きます。
       
       ◆最後に作成したスクラップ範囲
         Ctrl+1でスクリーンショットを表示し、0を押すと最後に作成したスクラップの範囲を切り抜きます。
         
    クリックキャプチャー
       ショートカットキーを使用せずにマウスだけでスクラップ生成が行えます。
       画面内のどの部分をクリックするか設定し、マウスでクリックするとキャプチャー準備に入ります。
       オプションにて設定可能です。
